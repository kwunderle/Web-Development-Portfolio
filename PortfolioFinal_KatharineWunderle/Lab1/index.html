HTML

HTML stands for Hyper Text Markup Language and is the standard way a web page is built. Physicist Tim Berners-Lee proposed and developed the first version in the early 1990s and it has had several different versions since then. HTML 4.01 was the official standard for many years with the implementation of tables, style sheets, and Javascript. The language is composed of different elements represented by tags. The start and end tags organize and differentiate the content in-between them. The tags are enclosed by angle brackets. These elements can be further defined with different attributes that give them a specific look or function. HTML allows users to structure these elements into a fully customized web page.

HTML5

HTML5 is the fifth and most recent version of HTML. This new markup language is the result of a collaborative effort made by the Web Hypertext Application Technology Working Group and the World Wide Web Consortium. First introduced to the public in 2008, the additions of single tags for audio, video, and interactive capabilities made it much more versatile in our digital age. The use of scalable vector graphics allows the user to easily view the web page on a desktop, smartphone, tablet, or any size screen comfortably. Many new elements and attributes were added, while some were altered, redefined, or completely eliminated. With this new iteration of the language, it has never been easier to design a diverse and flexible web page.
